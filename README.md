# 🏋️ APEX Fitness

A modern, responsive fitness and gym website built using **React**, **TypeScript**, and **Vite**. The website is designed to showcase a premium fitness center with a clean UI, engaging animations, and a seamless user experience.

## 🚀 Features

- Modern and responsive design
- Hero section with call-to-action
- Services section
- Trainers section
- Membership/Pricing plans
- Testimonials
- Contact form
- Smooth scrolling animations
- Mobile-friendly layout
- Fast performance using Vite

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Framer Motion
- GSAP
- Lucide React
- Supabase (if configured)

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/sigilikushala-rgb/APEX-Fitness.git
```

Navigate to the project directory:

```bash
cd APEX-Fitness
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
